# Remotion

Build videos with React – write video creation code instead of using a GUI.

## What You Can Build

- **Animated explanations** – Code-driven animations
- **Data visualizations** – Animated charts and graphs
- **Custom transitions** – Unique effects and compositing
- **Batch videos** – Generate hundreds of videos with variations
- **Motion graphics** – Complex animations and effects

## Why Use It

- **Programmatic** – Version control your videos
- **Reproducible** – Same input = same output
- **Customizable** – Full control over every frame
- **Interactive** – Preview and adjust in real-time
- **Export options** – MP4, WebM, GIF, sequences

## React Skills

Uses React components to define video structure and animation.

## Perfect For

- Video production automation
- Personalized video generation
- Intro sequences and branding
- Educational video creation
- Marketing video production at scale

## Getting Started

```bash
npm create video
```
