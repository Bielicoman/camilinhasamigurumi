# Professional Photo Editor

Edit photos with professional photography standards. Process batch images, apply color grading, and save optimized versions.

## What It Does

📸 **Photo Processing**
- Batch process multiple images at once
- Support for RAW, JPG, PNG, TIFF
- Lossless quality preservation
- Automatic rotation and orientation fix

🎨 **Professional Adjustments**
- Exposure correction
- White balance adjustment
- Contrast and clarity enhancement
- Saturation and vibrance control
- Shadows and highlights recovery
- Sharpening and noise reduction
- Color grading presets

🏷️ **Batch Management**
- Process entire folders
- Smart naming (original_EDITED.ext)
- Preserve EXIF metadata
- Organize by date or custom structure
- Progress tracking

## How to Use

### Step 1: Prepare Your Files
```
/raw-photos/
  ├── photo-001.raw
  ├── photo-002.jpg
  └── photo-003.png
```

### Step 2: Set Up Edit Profile
Choose editing style:
- **Portrait** – Skin tone enhancement, soft glow, detail preservation
- **Landscape** – Enhanced colors, clarity, shadow recovery
- **Product** – Neutral white balance, maximum detail, vibrant colors
- **Street** – High contrast, color pop, warm tones
- **Moody** – Low saturation, cool tones, dramatic lighting
- **Bright & Clean** – High exposure, minimal shadows
- **Custom** – Define your own adjustments

### Step 3: Process Images
```
Input folder: /projects/shoot-2024-03/raw
Output folder: /projects/shoot-2024-03/edited
Edit preset: Portrait
Apply to: All images
```

### Step 4: Review & Export
- Quick preview of before/after
- Adjust settings if needed
- Export edited images
- Generate contact sheet/index
