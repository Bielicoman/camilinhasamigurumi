# Schedule

Create and manage scheduled tasks that run automatically on a recurring schedule.

Use the scheduled-tasks MCP to create recurring tasks, one-time tasks at a specific time, or ad-hoc tasks that run manually.

## Common Use Cases

- **Recurring tasks**: Daily standup, weekly reports, hourly monitoring
- **One-time tasks**: Reminders, future automation, scheduled actions
- **Cron syntax**: Standard 5-field cron expressions for precise timing

## Commands

- `Create a scheduled task` – Set up recurring automation with cron expressions
- `Schedule a one-time task` – Run once at a specific time, then disable
- `List all scheduled tasks` – View all active and inactive scheduled tasks
- `Update a scheduled task` – Modify schedule, prompt, or enable/disable

## Examples

- "Check my inbox every morning at 9am"
- "Run a weekly report every Friday at 5pm"
- "Remind me in 5 minutes to check something"
- "Deploy at 2am UTC every day"
