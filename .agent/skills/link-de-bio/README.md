# Skill: Link de Bio

Cria páginas de bio links profissionais, personalizadas e prontas para publicar.

## Estrutura dos arquivos

```
link-de-bio/
├── SKILL.md          # Definição da skill
├── prompt.md         # Prompt padrão para copiar/colar
└── README.md         # Este arquivo
```

## Instalação

1. Copie a pasta `link-de-bio` para `.agents/skills/`
2. Reinicie o Claude
3. A skill aparecerá disponível com `/link-de-bio`

## Uso rápido

Use o comando padrão em `prompt.md`:

```
/link-de-bio
```

Depois preencha as informações solicitadas e o Claude gera seu HTML completo.

## Recursos

✅ Design responsivo (mobile e desktop)
✅ HTML puro, sem dependências
✅ Fácil de personalizar depois
✅ Pronto para Vercel, GitHub Pages, ou qualquer hosting
✅ Suporte a foto de perfil
✅ Links ilimitados
✅ Estilos personalizáveis
