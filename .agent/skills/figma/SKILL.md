# Figma Integration

Connect Claude Code with Figma to enable design-to-code workflows.

## Features

- **Design context** – Extract design metadata from Figma
- **Code Connect** – Map Figma components to code components
- **Screenshot generation** – Generate screenshots of designs
- **Component inspection** – Analyze Figma components and variables
- **Design system** – Manage design tokens and variables

## Common Workflows

1. **Design Review** – Get feedback on Figma designs
2. **Design-to-Code** – Convert Figma designs to React/Vue/etc
3. **Component Mapping** – Link Figma components to code
4. **Design System** – Extract and sync design tokens

## Getting Started

- Have Figma design file open
- Get Figma file URL with node IDs
- Use design context extraction to analyze components

## Perfect For

- Teams using Figma + code
- Design system management
- Rapid prototyping
- Component-driven development
- Design quality assurance

## Integration

Figma metadata flows into code generation, ensuring designs stay in sync with implementation.
