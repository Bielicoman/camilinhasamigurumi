# Claude API

Work with the Claude API and Anthropic SDKs to build AI applications.

## Available SDKs

- **Python SDK** – `pip install anthropic`
- **JavaScript SDK** – `npm install @anthropic-ai/sdk`
- **Agent SDK** – For building custom agents and extending Claude Code

## Common Patterns

### Chat Completions
Build conversational applications using the Messages API.

### Vision
Process images with Claude's vision capabilities.

### Tool Use
Implement function calling and agentic workflows.

### Streaming
Handle real-time response streaming for better UX.

## Authentication

Use API keys stored securely. Never hardcode credentials in code.

- Set `ANTHROPIC_API_KEY` environment variable
- Or pass explicitly when initializing the client

## Getting Started

Choose your language and start building!
