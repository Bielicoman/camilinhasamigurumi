## How to Use

### Phase 1: Log Conversion (Starting Point)
Convert log footage to viewable color space using technical LUT, then apply creative grading.

### Phase 2: Primary Grading
Adjust entire image:
- Exposure (gain/lift)
- Contrast (gamma)
- Whites & blacks (output levels)
- Color balance (shadows, mids, highlights)
- Saturation (overall and by color)

### Phase 3: Secondary Grading
Target specific colors/tones:
- Skin tones enhancement
- Sky color correction
- Foliage color pop
- Isolate and adjust specific hues
- Local adjustments by luminance

### Phase 4: Finishing
- DaVinci Resolve: Use nodes, curves, power windows
- Premiere Pro: Lumetri Color panel, adjustment layers
- Final Cut Pro: Color Board, curves, masks
- Export color-graded footage with LUTs

## LUT Formats & Compatibility

📦 **Universal LUT Formats**
- `.cube` – DaVinci Resolve, Premiere, After Effects, Nuke
- `.look` – DaVinci Resolve native
- `.3dl` – Universal (Autodesk)
- `.lut` – Varying formats
- `.dat` – Some specialized software

⚙️ **Software-Specific**
- **Resolve** – Native .cube, .look, uses Fusion for advanced
- **Premiere** – .cube LUTs in Lumetri panel
- **Final Cut** – .cube LUTs via extensions
- **After Effects** – .cube in Curves or third-party plugins
- **Nuke** – Native LUT nodes

## Step-by-Step Workflow

### 1. Prepare Your Footage
```
- Check color space (log or linear?)
- Read scopes to understand exposure
- Check for clipping in highlights/shadows
- Note white balance issues
```

### 2. Apply Technical LUT (if log footage)
```
- Load log-to-rec709 LUT as base
- This gives viewable image to work with
- Adjust levels/exposure as needed
```

### 3. Primary Color Correction
```
- Adjust overall exposure
- Set black point and white point
- Correct white balance
- Basic contrast
```

### 4. Apply Creative LUT (cinematic look)
```
- Choose film stock or style preset
- Stack on top of primary correction
- Adjust intensity (0-100%)
```

### 5. Secondary Corrections
```
- Fix skin tones if needed
- Adjust specific colors
- Local masks for targeted areas
```

### 6. Final Adjustments
```
- Saturation pass
- Luminance curve tweak
- Contrast final check
- Compare with reference footage
```

## Essential Techniques

✏️ **Curves Adjustment**
- Read the histogram
- Adjust shadows, midtones, highlights separately
- S-curve for contrast
- Reverse S-curve for flat look

🎯 **Color Wheels (Shadows/Mids/Highlights)**
- Shadow color cast → warm or cool
- Midtone saturation and hue
- Highlight color tone (often warm for cinematic)

🔍 **Scopes Usage**
- **Waveform** – See exposure distribution
- **Vectorscope** – Check color balance and saturation
- **Histogram** – Precise exposure levels

🎨 **Color Psychology**
- Warm tones = emotion, intimacy, danger
- Cool tones = distance, sadness, sci-fi
- Desaturated = period pieces, drama
- Saturated = happiness, energy, surrealism

## Pro Tips

1. **Grade in sequence** – Don't jump to creative grades
2. **Use references** – Compare to films you admire
3. **Take breaks** – Color blindness from staring at screen
4. **Check on flat monitor** – Not in dark room (reveals mistakes)
5. **Export and verify** – Check on different displays
6. **Create LUT exports** – Share consistent look across shots
